# Sistema de Pedidos

Prueba técnica desarrollada en Laravel 12.

## Pasos para instalar
1. **Clonar el proyecto:**
   ```bash
   git clone https://github.com/KL3YB3R/test-products.git
   cd test-products

2. **Instalar Dependencias:**
    composer install

2. **Ejecutar Migraciones:**
    php artisan migrate

2. **Iniciar el servidor:**
    php artisan serve
